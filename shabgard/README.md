# شب‌گرد: روستای خاموش 🏜🕯
اتاق فرار دونفره‌ی ترسناک با دنیای تصادفی، سه شب داستانی و چهار پایان.

## اجرا روی Vercel
1. این پوشه را در یک ریپوی GitHub بگذارید و در vercel.com با «Add New Project» وصل کنید
2. یا در ترمینالِ همین پوشه: `npx vercel --prod`
هیچ build لازم نیست — سایت استاتیک است.

## نحوه بازی
- هر دو بازیکن لینک را باز می‌کنند و **تماس صوتی** می‌گیرند
- یک نفر «کدِ شب» می‌سازد و به دیگری می‌گوید؛ هر دو همان کد را وارد می‌کنند (دنیای هر دو یکسان تولید می‌شود)
- یکی «کاوشگر» می‌شود (داخل روستا) و دیگری «رمزگشا» (ایستگاه)
- پیشرفت به‌صورت خودکار روی همان مرورگر ذخیره می‌شود؛ هر شب را می‌توانید یک روز بازی کنید
- کد جدید = معماها و جواب‌های کاملاً جدید
